## 🚀 התקנה והרצה

### דרישות מקדימות

- Python 3.8 ומעלה
- pip

### התקנה

```bash
# שכפול הפרויקט
git clone https://github.com/YOUR_USERNAME/new_project2.git
cd new_project2

# יצירת סביבה וירטואלית (אופציונלי)
python -m venv venv
venv\Scripts\activate   # Windows
# source venv/bin/activate   # Mac/Linux

# התקנת התלויות
pip install streamlit pandas plotly
```

### הרצה

```bash
streamlit run deshbord.py
```

האפליקציה תיפתח בדפדפן בכתובת: `http://localhost:8501`
